# 🎮 Hangman Game - Python

This is a simple text-based Hangman Game developed using Python. The game allows a user to guess a randomly selected word letter by letter within a limited number of attempts.

The program provides real-time feedback, tracks guessed letters, and displays the current progress of the word using underscores.

---

## 🚀 Features

* Random word selection
* Maximum 6 incorrect attempts
* Tracks guessed letters
* Displays word progress (`_ _ _ _`)
* Win/Lose condition messages

---

## 🛠️ Technologies Used

* Python
* Random Module
* Basic Loops & Conditions

---

## ▶️ How to Run

1. Install Python
2. Clone this repository
3. Run the file:

```
python hangman.py
```

## 📷 Sample Output
